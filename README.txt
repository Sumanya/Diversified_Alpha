Description

Diversified Alpha
Implementing Trend trading strategy with Theta decay
Implementing Index arbitrage on NIFTY and it's constituents 
Calculating KPIs to built a diversified portfolio

Installation
--------------
1. Install Python 3.6 - https://www.python.org/downloads/ 
2. Unzip files to local drive in desired folder (example: C:\sumanya_project). 
3. Open cmd prompt / shell.
4. Navigate to created folder.
5. Install requirements:
   Install pandas,numpy and matplotlib packages.
   Install talib and NSEpy moduses using pip install
6. Run “Portfolio.py” by typing Python Portfolio.py in command prompt/shell.
